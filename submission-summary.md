# Submission Summary

## 1. 서비스명

멍케어노트 MCP

## 2. 한 줄 소개

반려견의 음식 섭취, 일상 상태, 증상 표현, 사진 관찰 기록을 정리하고 보호자의 다음 관리 행동과 동물병원 상담 준비를 돕는 MCP 서버입니다.

## 3. 문제 정의

보호자는 반려견이 평소와 다른 행동을 보일 때 어떤 정보를 기록해야 하는지, 어느 정도로 주의해야 하는지, 동물병원에 무엇을 전달해야 하는지 판단하기 어렵습니다. 특히 음식 섭취 사고, 식욕 저하, 무른 변, 구토, 무기력 같은 일상 신호는 대화 속에 흩어지기 쉽고 병원 상담 시 핵심 정보가 빠질 수 있습니다.

## 4. 해결 방법

멍케어노트 MCP는 보호자의 자연어 입력을 MCP tool로 구조화합니다. 음식 위험도 확인, 일상 상태 위험도 분류, 오늘의 관리 행동 추천, 증상 표현 정리, 동물병원 상담용 요약, 지역 기반 동물병원 후보 안내, 사진 관찰 기록을 제공합니다.

이 서비스는 질병을 판단하거나 치료 방법을 제시하지 않습니다. 보호자가 반려견 상태를 더 잘 기록하고, 위험 신호가 있으면 수의사 상담으로 이어지도록 돕는 보조 도구입니다.

## 5. MCP Tool 목록

- `check_food_safety`: 반려견이 먹은 음식의 위험도를 확인하고 보호자 행동을 안내합니다.
- `analyze_daily_status`: 식욕, 변, 구토, 활동량 등 오늘 상태를 기반으로 위험도를 분류합니다.
- `recommend_daily_care`: 위험도와 주요 증상을 바탕으로 식단, 간식, 물 섭취, 산책, 휴식을 추천합니다.
- `create_vet_visit_summary`: 동물병원 상담 시 보여줄 수 있는 증상 요약문을 생성합니다.
- `find_nearby_animal_hospitals`: 지역명 기반으로 동물병원 후보를 안내합니다.
- `classify_pet_symptom`: 보호자 자연어 증상을 표준 증상명과 카테고리로 정리합니다.
- `record_pet_photo_observation`: 변/피부 사진 기록과 보호자 관찰 내용을 바탕으로 이상 징후를 정리합니다.

## 6. 사용 데이터

- 음식 위험도 규칙: MVP rule-based 로컬 규칙
- 일상 상태 위험도 규칙: MVP rule-based 로컬 규칙
- 동물병원 데이터: 공공데이터포털 동물병원 조회 API URL 사용 가능
- 동물병원 fallback 데이터: `src/data/animalHospitals.sample.json`
- 증상 사전 fallback 데이터: `src/data/symptomDictionary.sample.json`
- 사진 기록 데이터: `src/data/photoRecords.json`

공공데이터 API는 `PUBLIC_DATA_SERVICE_KEY`와 `USE_PUBLIC_DATA_API=true`가 설정되면 실시간 호출을 시도합니다. API 키가 없거나 호출이 실패하면 로컬 샘플 데이터로 fallback합니다.

## 7. 의료 안전 정책

- 모든 tool 응답에 안전 문구를 포함합니다.
- 안내는 진단이나 처방이 아닙니다.
- 이상 증상이 심하거나 지속되면 수의사 상담을 권장합니다.
- 혈변, 반복 구토, 매우 심한 무기력, 식욕이 전혀 없는 상태, 호흡 이상 등은 빠른 진료 권장을 명확히 안내합니다.
- 특정 질병명 확정, 약 복용 지시, 병원 방문 불필요 판단은 제공하지 않습니다.
- 사진 기능은 실제 이미지 진단이 아니라 사진 기록 및 이상 징후 기록 보조입니다.

## 8. 데모 시나리오

1. 보호자가 "강아지가 포도 한 알 먹었어. 몸무게는 11kg이야."라고 입력합니다.
2. `check_food_safety`가 위험 음식 섭취로 분류하고 빠른 진료 권장을 안내합니다.
3. 보호자가 "우리 강아지 6살 11kg인데 어제부터 밥을 반만 먹고 변이 묽어. 구토는 없어."라고 입력합니다.
4. `analyze_daily_status`가 식욕 감소와 무른 변을 근거로 위험도를 분류하고 오늘 관리 행동을 안내합니다.
5. 보호자가 "강아지가 밥을 안 먹고 축 처져 있어. 증상으로 정리해줘."라고 입력합니다.
6. `classify_pet_symptom`이 식욕저하, 무기력 같은 증상 표현으로 정리합니다.
7. 보호자가 변 사진 관찰 내용을 기록합니다.
8. `record_pet_photo_observation`이 사진 설명과 관련 증상을 바탕으로 이상 징후를 기록합니다.
9. 보호자가 "부산 진구 근처 동물병원 알려줘."라고 입력합니다.
10. `find_nearby_animal_hospitals`가 지역 기반 동물병원 후보와 방문 전 전화 확인 안내를 제공합니다.
11. 보호자가 "방금 증상들을 동물병원에 보여줄 수 있게 정리해줘."라고 입력합니다.
12. `create_vet_visit_summary`가 상담용 요약문과 수의사에게 물어볼 질문 목록을 생성합니다.

## 9. 향후 확장 계획

- 실제 배포 환경에서 공공데이터 API 키를 연결해 동물병원 조회 정확도 개선
- 지역명 검색을 좌표 기반 거리 검색으로 확장
- 사진 기록 기능에 외부 vision analyzer 연동
- 보호자별 반려견 프로필과 장기 기록 저장
- 반복 증상 추세 요약과 병원 상담 이력 관리
- PlayMCP 시연 흐름에 맞춘 tool chaining 개선
